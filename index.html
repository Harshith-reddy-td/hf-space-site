üîß Configuration Options

Option 1: Direct Redirect (Meta Refresh)

To automatically redirect visitors to your Hugging Face Space, replace the meta tag in the head section:

<meta http-equiv="refresh" content="0; url=https://huggingface.co/spaces/YOUR-USERNAME/YOUR-SPACE" />

Option 2: JavaScript Redirect

Add this script before the closing body tag:

<script>
    window.location.href = 'https://huggingface.co/spaces/YOUR-USERNAME/YOUR-SPACE';
</script>

Option 3: Embed Hugging Face Space (iframe)

Embed the Space directly on this page:

<iframe src="https://YOUR-USERNAME-YOUR-SPACE.hf.space" width="100%" height="800px" style="border:none;"></iframe>

üìù Steps to Map Your Hugging Face Space

  1. Create or identify your Hugging Face Space (e.g., https://huggingface.co/spaces/username/spacename)
  2. Edit this index.html file in the repository
  3. Add one of the redirect methods above with your Space URL
  4. Commit the changes
  5. Wait 1-2 minutes for GitHub Pages to rebuild
  6. Visit https://harshith-reddy-td.github.io/hf-space-site/ to see your Space

üîó Custom Domain (Optional)

To use a custom domain:

  1. Go to repository Settings ‚Üí Pages
  2. Enter your custom domain in the "Custom domain" field
  3. Add DNS records with your domain provider:
    • For apex domain (example.com): A records pointing to GitHub IPs
    • For subdomain (www.example.com): CNAME record pointing to harshith-reddy-td.github.io
  4. Enable "Enforce HTTPS" after DNS propagates
